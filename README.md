# Traveo – AI Travel Companion

## 🧭 Overview  
**Traveo** is an AI-powered chatbot agent designed to help users plan trips smartly and efficiently.  
It offers personalized destination suggestions, itinerary planning, transport and accommodation tips, and real-time travel guidance—all for free—deployable on IBM Cloud Lite using watsonx.ai Prompt Lab and IBM Granite foundation models.

---

## ✨ Features  
- **Conversational AI** — Chat-based assistance for trip planning, destination discovery, budgeting, and more.  
- **Built on Free Tier** — No paid IBM Cloud resources required.  
- **Prompt-engineered** — Utilizes optimized system prompts for accurate and helpful travel responses.  
- **Live Web or API Use** — Deployable as a webchat widget, API endpoint, or directly via IBM watsonx.ai Prompt Lab.

---

## 🚀 Usage  

You can interact with Traveo via:

- **IBM watsonx.ai Prompt Lab** (test prompts directly)  
- **Deployed Web Chat Widget** (if integrated)  
- **REST Endpoint** (send requests via `curl`, Postman, or your own app)

---

### 🌐 Web Chat Preview (if deployed)  
1. Open the provided web chat or embed-page link in your browser.  
2. Start a conversation: Ask about destinations, budgets, travel tips, or packing suggestions.

---

## 💡 Example Prompts  
1.“Do you have a preferred destination in mind?”

2.“What’s your estimated travel budget?”

3.“How long will your trip be?”

4.“Are you looking for flights, hotels, or both?”

## 📌 How to Run or Deploy

1. Log in to IBM Cloud Lite: https://cloud.ibm.com  
2. Launch **watsonx.ai Studio**  
3. Create a new AI Agent (Traveo)  
4. Configure agent instructions (introduce Traveo as your travel assistant)  
5. Add example questions like destination, budget, duration  
6. Test in the **Preview** panel  
7. Deploy via:
   - Web snippet (chat widget)
   - Streamlit app
   - Custom web UI  
8. Share the live agent with users for feedback

## ✅ Do’s and ❌ Don’ts

### ✅ Do:
- Ask about destinations, trip durations, cost estimates, or itinerary ideas  
- Use clear, conversational questions  
- Interact via Web Chat, Preview, or API interface  

### ❌ Don’t:
- Enter personal or financial information  
- Rely on the agent for visa, medical, or emergency decisions  
- Expect memory of past conversations (unless explicitly extended)
